# 图片转PDF工具 / Image to PDF Tool

一个简洁优雅、功能强大的图片转PDF工具，支持将多张图片合并转换为PDF文件。

A clean and powerful tool to convert multiple images into a single PDF file.

---

## ✨ 功能特点 / Features

- **多图合并 / Batch Merge** - 支持批量添加图片，合并为一个PDF文件 / Add multiple images and merge them into one PDF
- **拖拽添加 / Drag & Drop** - 直接拖拽图片到窗口即可添加 / Simply drag images into the window to add
- **排序功能 / Sort Function** - 支持上移/下移调整图片顺序 / Move items up/down to adjust order
- **预览功能 / Preview** - 点击图片可预览缩略图 / Click images to preview thumbnails
- **撤销/重做 / Undo/Redo** - 支持操作历史记录，可撤销/重做 / Operation history with undo/redo support
- **批量重命名 / Batch Rename** - 可批量修改列表中图片的显示名称 / Batch modify display names in the list
- **快捷键支持 / Shortcuts** - Ctrl+A全选、Ctrl+I反选、Delete删除、Ctrl+Z撤销 / Ctrl+A (select all), Ctrl+I (invert), Delete (remove), Ctrl+Z (undo)
- **PDF设置 / PDF Settings** - 可调整图片质量、选择PDF页面大小（A4/A3/Letter）/ Adjust image quality, select page size (A4/A3/Letter)
- **自动保存 / Auto Save** - 自动保存上次的设置和图片列表 / Automatically save last settings and image list
- **极简界面 / Minimal Interface** - 无边框设计，自定义标题栏，操作流畅 / Borderless design with custom title bar, smooth operation
